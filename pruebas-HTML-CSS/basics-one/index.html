HTML

HTML son las siglas de Hyper Text Mekup Lenguage, que puede traducirce como lenguaje de marcas o marcado de hipertexto.

El lenguaje HTML, se emplea para crear las páginas web.Es muy facil ver el codigo HTML de una página web, la opción exacto cambia dependiendo del navegador y la versión que se utiliza pero suelen utilizar un nombre similar

HTML se compone de etiquetas que se escriben entre simbolos menor que y mayor que.

Historia de HTML

Los inicios del lenguaje HTML se remontan al año 1990. cuando TIM Berners-Lee creó la primera página web.

Versiones de HTML

Tim Berners-Lee definió la primera versión de HTML en el año 1190.
En la actualidad la ultima versión de HTML es HTML$, HTML5.
